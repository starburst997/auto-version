Reference

Inputs and outputs for the action

üì• Inputs

  • main-branch
    Production branch name
    Default: main
  • dev-branch
    Development branch name
    Default: dev
  • default-version
    Starting version if no tags exist
    Default: 1.0.0
  • update-major-minor
    Update floating tags (v1, v1.2)
    Default: false
  • git-user-name
    Git committer name
    Default: github-actions[bot]
  • git-user-email
    Git committer email
    Default: github-actions[bot]@users.noreply.github.com

üì§ Outputs

  • version
    Calculated semantic version without 'v' prefix
    Example: 1.2.3-dev
  • tag
    Git tag with 'v' prefix
    Example: v1.2.3-dev

‚úÖ Requirements

  • fetch-tags: true
    Required in checkout step to access version tags
  • contents: write
    Required permission to push tags to repository
